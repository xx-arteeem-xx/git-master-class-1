8. Рассказываю о том, что такое область видимости git. Показываю, как можно добавить в нее файлы, как удалить из нее файлы 
``` PowerShell
git status
git add . # Добавить все файлы в директории 
git status
git reset index.html # Удалить конкретный файл из области видимости
git status
git add index.html # Добавить конкретный файл
git status
```
9. Делаем первый коммит
``` PowerShell

git status
git log # Команда для просмотра истории коммитов
git show hash_commit # Посмотреть инфо о конкретном коммите
git commit --amend -m "Новый комментарий" # Переназвать коммит (не рекомендуется - опасно)
```
